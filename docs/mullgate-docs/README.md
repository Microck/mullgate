# mullgate-docs

Internal docs-site application for publishing the public Mullgate documentation.

This README is for maintainers working on the site, not for Mullgate operators looking for setup or usage instructions.

## Local development

```bash
pnpm dev
```

## Checks

```bash
pnpm lint
pnpm types:check
pnpm build
```
